## 相关命令

```sh
sudo apt install -y ruby-full
gem update bundler

bundle clean --force
bundle install
bundle update

jekyll serve --force-polling --trace --livereload -P 4000 -H 0.0.0.0
```

注：WSL下不加`--force-polling`会卡死整个WSL，需要重启系统才能解决
